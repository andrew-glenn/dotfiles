Personal dotfiles — shell, tmux, neovim, ghostty, git, and assorted CLI configs.

Top-level dirs map to tools: `zsh/`, `tmux/`, `neovim/`, `vim/`, `ghostty/`,
`ssh/`, `git_templates/`, `ptpython/`, `cfnlint/`, `yamllint/`, `fonts/`,
`xenv/`. Helper `scripts/` and a `q0-keymap.json` (ZSA keyboard layout).

Install via `./deploy.sh` (symlinks into `$HOME`). `.gitmodules` pulls in
external configs. Edit in place and re-run deploy if needed.
